Document Structure

<!DOCTYPE> - Document Type Declaration

Defines the document type and version of HTML.

<!DOCTYPE html>

<html> - Root Element

The root element of an HTML document.

Attributes:
  • lang - Specifies the language of the document
<html lang="en">
  <!-- Content goes here -->
</html>

<head> - Document Head

Contains meta information about the document.

<head>
  <title>Page Title</title>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

<body> - Document Body

Contains the visible content of the document.

<body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>

<h2>Heading 2</h2>

<p> - Paragraph

Defines a paragraph of text.

<p>This is a paragraph of text. It can contain <strong>bold text</strong> and <em>italic text</em>.</p>

This is a paragraph of text. It can contain bold text and italic text.

<a> - Anchor (Link)

Creates a hyperlink to other web pages, files, or locations.

Attributes:
  • href - Specifies the URL of the link
  • target - Specifies where to open the linked document
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
Visit Example.com

<ul>, <ol>, <li> - Lists

<ul> defines an unordered list, <ol> defines an ordered list, and <li> defines list items.

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
  • First item
  • Second item
  • Third item
  1. First item
  2. Second item
  3. Third item

Multimedia

<img> - Image

Embeds an image in the document.

Attributes:
  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image
  • width, height - Set the dimensions
<img src="https://via.placeholder.com/150" alt="Placeholder image" width="150" height="150">

<audio> - Audio

Embeds sound content in the document.

Attributes:
  • src - Specifies the audio file
  • controls - Adds audio controls
  • autoplay - Starts playing automatically
  • loop - Repeats the audio
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

<video> - Video

Embeds video content in the document.

Attributes:
  • src - Specifies the video file
  • controls - Adds video controls
  • width, height - Set the dimensions
  • autoplay - Starts playing automatically
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Forms & Input

<form> - Form

Creates an HTML form for user input.

Attributes:
  • action - Specifies where to send form data
  • method - Specifies HTTP method (GET or POST)
<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

<input> - Input Field

Creates an input field where users can enter data.

Attributes:
  • type - Specifies the input type (text, password, email, etc.)
  • name - Specifies the name of the input
  • placeholder - Shows hint text
  • required - Makes the field mandatory
<input type="text" name="username" placeholder="Enter username" required>
<input type="password" name="password" placeholder="Enter password" required>
<input type="email" name="email" placeholder="Enter email" required>
<input type="number" name="age" min="1" max="100">
<input type="date" name="birthdate">








Tables

<table> - Table

Defines an HTML table.

<table>
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$100</td>
  </tr>
  <tr>
    <td>February</td>
    <td>$80</td>
  </tr>
</table>
Month Savings
January $100
February $80

Semantic Elements

<header> - Header

Represents introductory content or navigation links.

<header>
  <h1>Website Title</h1>
  <nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
  </nav>
</header>

<footer> - Footer

Represents the footer of a document or section.

<footer>
  <p>Copyright © 2023</p>
  <p><a href="mailto:info@example.com">Contact Us</a></p>
</footer>

<article> - Article

Represents self-contained content.

<article>
  <h2>Article Title</h2>
  <p>Article content goes here...</p>
</article>

Interactive Elements

<details> & <summary> - Details Disclosure

Creates a disclosure widget that can be toggled open and closed.

<details>
  <summary>Click to expand</summary>
  <p>This is the hidden content that appears when you click the summary.</p>
</details>
Click to expand

This is the hidden content that appears when you click the summary.

<dialog> - Dialog Box

Represents a dialog box or modal window.

<dialog id="myDialog">
  <p>This is a dialog box!</p>
  <button id="closeDialog">Close</button>
</dialog>
<button id="openDialog">Open Dialog</button>

<script>
  document.getElementById('openDialog').addEventListener('click', function() {
    document.getElementById('myDialog').showModal();
  });
  
  document.getElementById('closeDialog').addEventListener('click', function() {
    document.getElementById('myDialog').close();
  });
</script>

This is a dialog box!

Global Attributes

Common Global Attributes

These attributes can be used on any HTML element.

Attribute Description Example
class Specifies one or more class names for an element <p class="highlight">Text</p>
id Specifies a unique id for an element <div id="header">...</div>
style Specifies inline CSS styles <p style="color:red;">Red text</p>
title Provides additional information about an element <abbr title="HyperText Markup Language">HTML</abbr>
data-* Used to store custom data private to the page <div data-user-id="123">User</div>

This guide covers the most commonly used HTML elements and attributes. HTML5 introduced many semantic elements that improve accessibility and SEO.
